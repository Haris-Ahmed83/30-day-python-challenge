# 30-Day Python Coding Challenge

Welcome to my 30-day Python coding challenge! This repository contains a collection of professional Python projects, one for each day, ranging from CLI tools to web applications and data processing pipelines.

## Project Structure

Each day's project is located in its own directory: `day-XX-project-name/`.
